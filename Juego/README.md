# Juego Adivina el Número

Descripción
Este proyecto consiste en un juego interactivo desarrollado con HTML, CSS y JavaScript donde el usuario debe adivinar un número aleatorio generado por el sistema.

El jugador tiene un máximo de 10 intentos para adivinar el número correcto. Después de cada intento, el sistema proporciona pistas indicando si el número ingresado es mayor o menor que el número secreto.

---

# Funcionalidades

- Generación de número aleatorio entre 1 y 100
- Límite de 10 intentos
- Mensajes de ayuda para el jugador
- Historial de números ingresados
- Opción para reiniciar el juego

---

# Lenguajes utilizados

- HTML
- CSS
- JavaScript

---

# Objetivo del proyecto
Practicar el uso de JavaScript para manipulación del DOM, eventos y lógica de programación básica.
